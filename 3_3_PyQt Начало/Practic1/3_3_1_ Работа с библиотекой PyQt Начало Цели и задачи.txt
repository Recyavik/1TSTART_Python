Сегодня на уроке концептуально:
    - базовые методы PyQt для создания пользовательского интерфейса
    - QApplication: этот класс представляет главное приложение PyQt
   	- QWidget: является базовым классом для всех элементов пользовательского интерфейса в PyQt
    - QPushButton: представляет кнопку в пользовательском интерфейсе
    - QLabel: это виджет, используемый для отображения текста или изображений
