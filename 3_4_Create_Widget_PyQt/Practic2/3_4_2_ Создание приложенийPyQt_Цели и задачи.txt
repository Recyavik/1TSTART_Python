Тема: "Создание своего приложения и добавление виджетов на экран"

Сегодня на уроке концептуально:
    - методы, классы, виджеты PyQt для создания пользовательского интерфейса:
    - QApplication: этот класс представляет главное приложение PyQt
   	- QWidget: является базовым классом для всех элементов пользовательского интерфейса в PyQt
    - QLayout: это класс, который управляет расположением виджетов внутри других виджетов
        •	QVBoxLayout,
        •	QHBoxLayout,
        •	QGridLayout)
    - QPushButton: представляет кнопку в пользовательском интерфейсе
    - QLabel: это виджет, используемый для отображения текста или изображений
    - QLineEdit: это текстовое поле, в котором пользователь может вводить текст
    - QComboBox: это выпадающий список из предварительно заданного набора
    - QFileDialog: это диалоговое окно загрузки файлов
    - Сигналы и слоты: PyQt позволяют объектам взаимодействовать и обрабатывать события
