Практика 3. "ООП, Классы"
 Тема:
"ООП - объектно-ориентированное программирование. Знакомство с классами.
Свойства атрибутов экземпляров класса. Наследование"

Сегодня на уроке концептуально:
    - Класс
    - Атрибут класса
    - Экземпляр класса
    - Свойство @property
    - декораторы (@property и @name_setter) для работы с приватными атрибутами
    - Наследование - парадигма ООП
Сегодня на практике мы научимся:
    - Создавать свойства класса через декоратор @property
    - Использовать метод @name_setter для установки значения свойств
    - Удалять свойства экземпляров класса через декоратор @deleter
    - Создавать базовые(родительские) классы и дочерние классы,
        которые наследуют свойства и методы родительских классов
