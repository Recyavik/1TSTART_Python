Тема: "Работа с картинками и библиотекой Pillow"

Сегодня на уроке концептуально:
    - Библиотека Pillow основанная на библиотеке PIL (Python Imaging Library)
    - изображение, размер изображения, пиксели
    - цветовая палитра изображения
    - фильтры изображения


Сегодня на практике мы научимся c помощью библиотеки pillow:
    - загружать и сохранять графические файлы и размещать их на форме
    - изменять размеры изображения
    - рисовать сверху изображений
    - объединять изображения склеивая их
    - применять различные цветовые фильтры для изменения палитры
    - конвертирование изображений в чёрно-белое

pip install pillow
pip install pillow --upgrade
