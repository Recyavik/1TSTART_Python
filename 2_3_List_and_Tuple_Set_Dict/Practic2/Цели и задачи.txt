Сегодня на уроке: Методы строк.

 Методы строк — это функции, которые можно вызывать для объектов строки (str) в Python.
Они позволяют выполнять различные операции над строками, такие
как изменение регистра, поиск подстроки, замена части строки и т.д.

Научимся:
    upper() — преобразовывать строку в верхний регистр.
    title() - преобразовывать строку по типу CamelCase
    lower() — преобразовывать строку в нижний регистр.
    capitalize() — преобразовывать первый символ строки в верхний регистр и все остальные в нижний регистр.
    strip() — удалять пробелы в начале и конце строки.
    replace() — заменять вхождение одной строки другой строкой.
    find() — искать индекс первого вхождения подстроки в строке.
    split() — делить строку на части по указанному разделителю и возвращает список частей.
    join() - соединять строки из частей списка через разделитель
    len() — возвращать длину строки.
    isdigit() — возвращать True, если строка состоит только из цифр, иначе False.
    startswith() — возвращать True, если строка начинается со вложенной строки
