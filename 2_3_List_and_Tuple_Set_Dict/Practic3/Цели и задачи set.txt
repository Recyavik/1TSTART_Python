Сегодня на Уроке: "Множества"

    Множество — обозначает набор или совокупность неких уникальных элементов,
    которые обладают каким-то общим свойством и сгруппированных под одним именем.
    Соответствующая структура данных в языке Python называется set.

Концептуально:
    - понятие set - множество
    - способы обращения к элементам множества
    - простые алгоритмы обработки множеств
    - дубликаты элементов множеств (возможны ли они?)
На примере практических задач мы научимся:
    - Научимся создавать множества
    - Добавлять, удалять, редактировать элементы множеств, если такое возможно
    - Проанализируем уникальность значений множеств
    - Научимся использовать множества для хранения случайных чисел
